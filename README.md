# unraid-templates

Private-use unraid Docker templates for lukeg01. Add this repo URL in unraid
under **Docker → Template repositories**, then the templates appear in
**Docker → Add Container → Template**.

- `finance-tool.xml` — personal expense tracker
  (image: `ghcr.io/lukeg01/finance-tool`, private — requires
  `docker login ghcr.io` on the unraid host).
